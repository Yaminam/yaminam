  A few notes so it renders perfectly:

  1. Theme is unified — everything (stats, streak, languages, trophies, activity graph, banner) uses one tokyonight
  palette and for-the-badge style, which is what makes it read as designed rather than assembled. Want a different vibe?
  Swap tokyonight → radical, gruvbox, midnight-purple, or catppuccin_mocha everywhere at once.
  2. Username check — your stats point at yaminam. The snake and activity graph also use yaminam; make sure that's your
  real profile handle (your visit-counter used shreyashtripathi, so double-check which one is the account).
  3. The snake animation isn't a live service — it needs a small GitHub Action in your profile repo (yaminam/yaminam).
  Create .github/workflows/snake.yml:

  name: Generate Snake
  on:
    schedule: [{ cron: "0 */12 * * *" }]
    workflow_dispatch:
  permissions: { contents: write }
  jobs:
    generate:
      runs-on: ubuntu-latest
      steps:
        - uses: Platane/snk@v3
          with:
            github_user_name: ${{ github.repository_owner }}
            outputs: dist/github-contribution-grid-snake-dark.svg
        - uses: crazy-max/ghaction-github-pages@v4
          with: { target_branch: output, build_dir: dist }
          env: { GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }} }
